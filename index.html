ShockFitting (SF)

This will be a modular shock fitting code to be plugged to arbitrary CFD codes

View project on GitHub

SF is a set of C++ libraries for performing shock fitting on unstructured meshes.

The code is the result of the collaboration among:

  • Prof. Renato Paciorri and Valentina De Amicis at University La Sapienza of Rome
  • Prof. Aldo Bonfiglioli at University of Basilicata
  • Dr. Andrea Lani at the Von Karman Institute for Fluid Dynamics.

For more info, you can check the wiki.
